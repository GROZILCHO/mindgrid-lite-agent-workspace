# Marketing Campaign Workflow

Use this workflow for campaign planning, landing pages, email sequences, ads, and content campaigns.

## Steps

1. Project Lead confirms campaign goal, audience, offer, channels, timeline, and constraints.
2. Strategy Agent defines campaign angle, message, and success criteria.
3. Content SEO Agent drafts campaign content and reviews search or content opportunities.
4. UX Design Agent structures landing pages or conversion flows if needed.
5. Technical Agent prepares implementation or publishing instructions.
6. QA Audit Agent reviews claims, links, consistency, and approval needs.
7. Documentation Agent records campaign status and next actions.

## Required Memory Updates

- `PROJECT_BRIEF.md` if campaign changes project direction
- `TASKS.md`
- `DECISIONS_LOG.md`
- `ISSUES_LOG.md`
- `NEXT_ACTIONS.md`
