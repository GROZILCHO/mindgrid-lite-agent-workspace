# SEO Content Audit Skill

## Purpose

Review content for search intent, clarity, structure, trust, and conversion.

## When To Use

Use for page copy, blog posts, landing pages, service pages, and content refreshes.

## Required Inputs

- Content or page URL notes.
- Target audience.
- Page goal.
- Target keywords, if known.

## Process

1. Identify the page intent and reader need.
2. Review title, headings, structure, and metadata.
3. Check clarity, completeness, evidence, and calls to action.
4. Flag duplicate, thin, unsupported, or confusing content.
5. Recommend practical improvements.

## Output Format

- Overall assessment.
- Priority findings.
- Suggested content changes.
- Metadata recommendations.
- Risks and open questions.

## Risks

- Over-optimizing for keywords.
- Making claims without evidence.
- Ignoring conversion and user clarity.
